## Notes

### Knowledge

- 树形 DP 的背包问题，本质上可以看作是对每个子树进行一次分组背包合并。
- 处理子节点状态时，需要特别注意循环顺序，避免状态被提前覆盖。
- Git 的 `-u` 参数可以建立本地分支与远程分支之间的 upstream 关系。
- 建立 upstream 后，后续可以直接使用 `git push` 和 `git pull` 进行同步。

### Engineering Notes

```bash
git push -u origin main
```bash
git push -u origin main
```

含义：

```txt
-u = --set-upstream
origin = 远程仓库名称
main = 远程分支名称
```

建立 upstream 之后，后续可以直接执行：

```bash
git push
git pull
```
